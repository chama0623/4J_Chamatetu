## Description
4Jで作成したミニゲーム「ちゃま鉄」のプロジェクトファイルです.実装した機能としては次に示すものがあります.

- 日本語の描画
- 年,月,季節の管理
- 動的かつ季節に応じたマップ描画
- プレイヤーの移動
- 物件駅,プラス駅,マイナス駅,カード駅
- 物件の購入操作,独占
- 決算と最終成績
- 借金時の物件自動売却
- カード(急行,特急,新幹線,サミット,ぶっとび,10億円,徳政令,豪速球)
