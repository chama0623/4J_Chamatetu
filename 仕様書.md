player構造体
name : 名前
money : お金(万円単位) 初期値1000
x : x座標
y : y座標
isBonby : 貧乏神フラグ
cardnum : カード枚数
cardlist : 所持カードid配列
cardoption : カードのオプション(破棄不可能フラグ,周遊残り回数)
(オプション : 絶好調フラグ)

price : 価格

name : 駅名
